Интерфейс - все в спецификаторе доступа public.
Полиморфизм - позволяет объединить разные типы данных в колекцию, и у этой колекции есть общий интрефейс, возможность делать разные реализации этому главному интерфейсу.  
Абстрактный класс - не имеет смысла в существования.
-------------------------------------------------
Интерфейс - это абстрактный класс только с pure virtual function
Пример:
template <typename T>
class Icollection
{
public:
	virtual void Add(T item) = 0;
};
template <typename T.
class Array : public Icollection<T>
{
	virtual void Add(T item)	
	{
		//Тело метода класса Add
	}
};
