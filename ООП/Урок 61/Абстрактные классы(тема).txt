Абстрактные классы - это классы, в котором нельзя создавать объекты.
pure virtual function - появление такой виртуальной функции сопровождается появлением асбтрактного класса.
pure virtual function - метод есть, но он пока не реализован.
Например:
virtual void Move() = 0; // Абстарктный метод либо pure virtual function
Только после того как все абстрактные методы будут реализованы в дочерних объектах, только тогда компилятор снимет пометку с класса что он абстрактный.
Шаблонный метод нельзя сделать виртуальным.
------------------------------------------------------------------------
Как сделать класс Абстрактным и еще в главном классе написать тело абстрактного метода:
Решение:
virtual void Move() = 0;
virtual ~Vehicle() = 0{};
