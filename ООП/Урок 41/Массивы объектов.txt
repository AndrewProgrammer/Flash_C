При создании массива объектов для каждого из объектов создается автоматически конструктогр по умолчанию.
Например: Date dates[3];
Последовательность отработки кода:
Source code - код cpp;
Препроцессор - это программа которая подготавливает главный код(cpp код) к компиляции, подключается также лексический анализатор а также макросы, чем больше макросов тем больше код в компиляторе будет отличатся от того кода который написан в cpp файле.
Единица трансляции - это один cpp файл со всеми его include.
Например - есть три хидера и три cpp, то компилятор вызывается 3 раза и 3 раза срабатывает cpp; 
Компилятор принимает выражения, а после отработки мы получаем последовательность токенгов.
